General

This api is small project so there is not very much to talk about so I'll be as short as possible. There are not many options as to how user can choose data from database, because you can take every record or just specific ones.

Choose records

Take every record from category table:
http://127.0.0.1:5000/category/

To take record which category_id=1 from category table, just pass the key and values, in query:
http://127.0.0.1:5000/category/category_id=1

To pass multiple values use & sign:
http://127.0.0.1:5000/category/category_id=1&3&7

To select records by multiple keys use coma , as a delimiter, records will appear in the order in which values were passed
http://127.0.0.1:5000/category/category_id=1&3&7,name=Animation&Comedy

If you want to select a record by passing string you have to use ' ' but only if u re passing 1 value:
http://127.0.0.1:5000/category/name='Action'
In case of multiple string values using ' ' will result in error
http://127.0.0.1:5000/category/name=Action&Comedy

Queries to every table works the same, you specify keys and values. Let's take film table for example. The film table has many different keys, which you can check by passing table_name to query /film/.
http://127.0.0.1:5000/film/
To take every record whose rental_duration is 6 we can pass:
http://127.0.0.1:5000/film/rental_duration=6
On this site you can check dvdrental database description and details.
